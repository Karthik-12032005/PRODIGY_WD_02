# ⏱ Stopwatch Web Application

A modern and responsive Stopwatch Web Application built using HTML, CSS, and JavaScript.

## 🚀 Features

* Start Stopwatch
* Pause Stopwatch
* Reset Stopwatch
* Record Lap Times
* Responsive Design
* Interactive Modern UI

## 💻 Technologies Used

* HTML5
* CSS3
* JavaScript

